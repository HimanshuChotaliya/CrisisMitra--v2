CrisisMitra v2 🚨

Live Demo ---> https://dev-pulse-interactive-platform-for.vercel.app/
Real-Time Disaster Relief Coordination Platform

🏆 Inspired by the 1st Place winning project at the JARVIS 24-Hour Hackathon

CrisisMitra v2 is the next-generation evolution of CrisisMitra, a disaster relief coordination platform designed to bridge communication between victims, volunteers, and authorities during emergencies.

Built with a stronger backend architecture using Flask (Python), the platform focuses on real-time coordination, offline-first accessibility, and efficient disaster response management in network-constrained environments.

Unlike traditional emergency systems that rely heavily on stable internet connectivity, CrisisMitra v2 is designed to remain functional even during communication breakdowns caused by floods, earthquakes, cyclones, or other large-scale disasters.

🌟 Key Features
🚨 Real-Time SOS Alerts

Victims can instantly raise SOS alerts with:

Live geolocation
Disaster descriptions
Emergency severity details

Alerts are visualized in real time on an interactive disaster response map for rapid situational awareness.

🗺️ Interactive Disaster Mapping

Built using mapping integration and real-time location tracking, the platform provides:

Live incident tracking
Volunteer movement visibility
Resource distribution monitoring
Geographic clustering of emergency requests

This helps coordinators prioritize rescue operations efficiently.

🙋 Volunteer Coordination System

Dedicated volunteer dashboards allow responders to:

View assigned missions
Track rescue priorities
Update task progress
Monitor nearby incidents

This creates structured coordination between field volunteers and emergency coordinators.

📊 Centralized Command Dashboard

Administrators and coordinators can monitor:

Active SOS alerts
Volunteer availability
Disaster zones
Resource allocation
Real-time emergency updates

The dashboard acts as a live command center during disaster situations.

🌐 Multilingual Accessibility

To ensure wider accessibility, the platform supports:

English
Hindi
Marathi

This helps users communicate effectively during emergencies regardless of language barriers.

📶 Offline-First Design

One of the core goals of CrisisMitra is resilience during connectivity failures.

Key features continue functioning even under poor or no internet conditions:

SOS registration
Local alert storage
Map interaction
Emergency reporting

Data automatically synchronizes once connectivity is restored.
